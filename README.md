# EAP Web Application

EAP Web Application is an internal manufacturing support system developed during an internship program to improve the monitoring and management of machine operations within the IC manufacturing process.

The project focused on redesigning and modernizing a legacy host-based system that had been used within the company for more than 20 years. The original system was limited in accessibility and user experience, so the application was transformed into a modern responsive web application that can be accessed through the company intranet across multiple devices.

The system was designed to provide a clearer and more efficient workflow for monitoring machine status, viewing production process information, and supporting operational management within the manufacturing environment.

---

## Responsibilities

- Designed and developed responsive user interfaces for manufacturing operation workflows
- Redesigned legacy system interfaces into a modern web application experience
- Developed front-end features and dynamic components using Angular and TypeScript
- Created UI/UX prototypes and layouts using Figma and Canva before implementation
- Optimized the application for desktop, tablet, and mobile platforms
- Participated in deploying and testing the application within the company environment
