# EAP Web Application

EAP Web Application is an internal web-based system developed for monitoring and managing machine status within the IC manufacturing process.  
This project was developed during an internship program to modernize the company’s legacy host-based system that had been used for over 20 years.

The system was redesigned into a responsive web application accessible through the company intranet, allowing users to monitor machine operations, process status, and production flow more efficiently across multiple devices.

---

## Features

- Machine status monitoring
- Process and production overview
- Responsive UI for desktop, tablet, and mobile
- Real-time status display interface
- Modernized UI/UX from legacy host system
- Internal intranet accessibility

---

## Technologies Used

- Angular 19
- TypeScript
- JavaScript
- HTML5
- LESS
- Tailwind CSS
- Figma
- Canva

---

## UI/UX Design

The interface and workflow were designed using Figma and Canva before implementation, focusing on usability, readability, and responsive user experience for operators and engineers.
